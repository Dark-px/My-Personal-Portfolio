# مستندات پروژه Parsa Portfolio (نسخه فارسی)

نسخه اصلی مستندات به زبان انگلیسی در این مسیر قرار دارد:

- `docs/README.md`

خوش اومدی. اين پوشه طوري طراحي شده که مجبور نشي براي هر تغيير کوچيک، کل کد رو بگردي.
مستندات به چند فايل جدا تقسيم شدن تا سريع به همون بخشي برسي که لازم داري.

## نقشه مستندات

1. `docs/01-SETUP.md`
   - راه اندازي پروژه، نصب، اجرا، متغيرهاي محيطي، ديپلوي

2. `docs/02-ARCHITECTURE.md`
   - معماري پروژه، جريان رندر، ساختار فولدرها، نقطه هاي مهم براي توسعه

3. `docs/components/PORTFOLIO_COMPONENTS.md`
   - تمام کامپوننت هاي بخش اصلي سايت (Hero, Projects, About, ...)
   - اينکه هر کدام کجاست، چه مي کند، چه بخشي قابل شخصي سازي است

4. `docs/components/BLOG_COMPONENTS.md`
   - معماري بخش Blog/Devlog و اتصال به Medium

5. `docs/components/UI_COMPONENTS_REFERENCE.md`
   - رفرنس کامل کامپوننت هاي UI موجود در پروژه (shadcn/radix)
   - دسته بندي شده بر اساس کاربرد

6. `docs/components/COMMON_COMPONENTS.md`
   - کامپوننت هاي عمومي مثل ErrorBoundary و Loading

7. `docs/components/CORE_APP_AND_INFRA.md`
   - مستندات App shell، pages، hooks، libها و integrationها

8. `docs/03-CUSTOMIZATION_PLAYBOOK.md`
   - نقشه سريع براي شخصي سازي پروژه بدون به هم ريختن ساختار
